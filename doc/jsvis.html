Visualizing Software Projects In JavaScript

Tim Disney

Visualization techniques have been used to help programmers deepen their understanding of large software projects. However the existing visualization techniques have focused on static languages where type information can be determined at compile-time. This project is a new visualization tool for the dynamic language JavaScript.
